Descripción del proyecto

Desarrollo de una plataforma web modular que convierte el proceso de colonoscopia en una fuente estructurada de información, generando valor para el paciente, enfermería, institución, médico y Tecnoquímicas. La construcción sigue un enfoque progresivo tipo MVP, priorizando la captura confiable de datos en fases tempranas y añadiendo análisis y automatización en fases posteriores.

Stack tecnológico: Next.js 16 · React 19 · TypeScript · Supabase (PostgreSQL + Auth) · Tailwind CSS · Vercel · WhatsApp Business API (Meta Cloud)

Condiciones generales

Cambios de alcance

Cambios que impliquen nuevas funcionalidades se cotizan como adiciones. Ajustes sobre lo definido en Fase 0 se atienden sin costo adicional.

Garantía post-entrega

30 días de garantía por bugs derivados del desarrollo, sin costo. No aplica para cambios de requerimientos ni errores de datos del cliente.

Tiempos estimados

Fase 0: 2 semanas · Fase 1: 4–5 semanas · Fases 2 y 3: 4 semanas c/u · Fase 4: 3 semanas. Total estimado: ~4 meses con dedicación completa.

Capacitación

Sesión de entrega y capacitación a usuarios administradores incluida en cada fase. Máx. 2 sesiones de 1 hora por fase.

Propiedad del código

El código fuente es propiedad del cliente una vez completado el pago total. Se entrega repositorio GitHub con documentación técnica básica.

Infraestructura

Hosting (Vercel) y base de datos (Supabase) son servicios de terceros a cargo del cliente. Costo estimado: USD $25–50 / mes según volumen.

IVA

Los valores expresados no incluyen IVA. De aplicar, se adicionará el 19% sobre los servicios de desarrollo según normativa tributaria vigente.

Pilotos entre fases

Se contempla una prueba piloto con usuarios reales al final de Fase 1 y al final de Fase 3, coordinada con el equipo de Tecnoquímicas.
